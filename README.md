# react-pos
A STRAKS Point-of-Sale system written in ReactJS for STRAKS Gate

## Installation
    git clone https://github.com/AndrewWelc/straks-react-pos.git
    cd react-pos/
    npm install
    
## Run
    npm start
    
## Build for Production
    npm run-script build
